Projects

Safe Route Calculator

We created a machine learning-driven app designed to offer safe travel route suggestions by analyzing crime data and dangerous area clusters. The app helps users, especially those navigating unfamiliar areas at night, by providing real-time insights into safer routes. We utilized technologies like Flask, Google Maps API, JavaScript, HTML/CSS, PHP, and Python to build the platform.

Covid-19 Mortality Analyzer

This project uses Python to create a machine-learning model that analyzes the probability of mortality of COVID-19 patients based on their various symptoms. The front end was created with HTML, CSS, and JavaScript to display the user's predicted results from their inputs. This project is currently in development.

Aerial Refueling Simulation

This project utilizes Unreal Engine 5 and Blender to model the Aerial Refueling process of Boeing's KC-46 and KC-135 aircraft. The simulation accurately depicts the functionality of the aircraft and the true-to-life aerial refueling process. We provided this simulation as a research tool to clients at the Air Force Life Cycle Management Center Future Tankers Program Office.

Credit Score Calculator and Analyzer

We trained a proprietary machine learning model that accurately predicts your credit score by analyzing a dataset of over 1000 data values related to the metric. After assessing the data, EduScore provides information on how you can improve your calculated credit score to be better positioned financially moving forward.

Virtual Reality Simulation of the Lunar South Pole

This project aims to utilize Unreal Engine 5 to create a highly realistic and immersive virtual model of the Lunar south pole in preparation for NASA's Artemis 3 mission. Our team collaborated with NASA and RHDO to provide this simulation as a research tool to demonstrate the various lighting and terrain effects of the moon.

Laser Eye Protection Simulation

This project utilized Unreal Engine 4 and C++ to create a simulation in both 2D and virtual reality to test the effects of harmful lasers and how they can be prevented with laser eye protection goggles. It also utilizes eye-tracking technology to safely and accurately model the dangerous effects of the laser on the eye.
